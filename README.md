# Blog

A simple blog API that allows users to

1. Register.
2. Create blog posts
3. Comment on the blog posts.

## Documentation

Checkout the [documentation](https://app.swaggerhub.com/apis-docs/Melvin1Atieno/blog-api/1.0.0#/)to make live requests.

## presiquites. 

1. An environment that can run `docker` and `docker-compose`


## Testing

1. Clone the repo
2. cd into repo directory
3. Buid image `docker-compose build`
4. Run container `docker-compose up -d`
5. Run tests `docker-compose run api bundle exec guard`


## Live testing

The endpoints can be tested live on the [documentation page](https://app.swaggerhub.com/apis-docs/Melvin1Atieno/blog-api/1.0.0#/)
